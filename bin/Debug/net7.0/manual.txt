3) HODNOCENI PODANI - pomoci numpadovych klaves 0, 1, 2 a 3
3 = Eso
2 = Tezky servis (mic je ve hre, ale maji problemi)
1 = Normal (normalni prijem)
0 = Zkazene podani

5) HODNOCENI PRIJMU - opet pomoci numpadovych klaves 0, 1, 2 a 3
3 = Nejlepsi (lze nahrat v klidu prstama)
2 = Dobry (nahravac uz musi bezet)
1 = Spatny (nahravac uz pouze spravuje, neda se dat hezka nahra)
0 = Dostal eso

6) TYP UTOKU - pomoci pismen L a S
L = Lob (kopecek)
S = Smec (tvrdy uder)

7) HODNOCENI UTOKU - pomoci numpadovych klaves 0, 1, 2 a 3
3 = Nejlepsi (rovnou na zem / polar odpalil mic pryc)
2 = Dobry (polar jen tak tak zvedl uder, vraci mic zadarmo zpet)
1 = Spatny (polar capl tak ze muze utocit zpet)
0 = Nejhorsi (primi bod pro soupere - utok do autu / dotek site / utok do bloku)
